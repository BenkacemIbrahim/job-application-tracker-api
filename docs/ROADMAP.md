# Roadmap

## Near Term

- Add refresh token support.
- Add email verification and password reset flow.
- Add API rate limiting and request throttling.
- Add database migrations for seed/admin bootstrap.

## Mid Term

- Introduce role-specific admin endpoints.
- Add search by company and keyword.
- Add soft delete and audit history for job updates.
- Add OpenTelemetry tracing and metrics dashboard.

## Long Term

- Multi-tenant support for organizations.
- Event-driven notifications (interview reminders, follow-ups).
- Analytics exports and BI-ready reporting endpoints.
