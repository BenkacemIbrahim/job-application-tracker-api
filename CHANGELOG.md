## [1.0.0] - 2026-02-19

### Added
- Spring Boot 3 API for job tracking with JWT auth.
- Role-based authorization (`USER`, `ADMIN`).
- CRUD endpoints for job applications with pagination, filtering, and sorting.
- Stats endpoint with JPQL aggregation.
- Global exception handling.
- PostgreSQL + Flyway migration support.
- Dockerfile and docker-compose production stack.
- Integration/unit tests and JaCoCo coverage reporting.
- GitHub readiness files (CI, templates, contributing, security, license).
